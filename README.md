# TruFont

TruFont is a font-editing application written with Python3, ufoLib, defcon and PyQt5.

Dependencies:

- [Python 3]
- [PyQt5]
- [fontTools]
- [robofab, python3-ufo3 branch]
- [defcon, python3-ufo3 branch]
- [ufo2fdk, python3-ufo3 branch]Â (optional)

Run:

`cd Lib && python3 -m defconQt`
